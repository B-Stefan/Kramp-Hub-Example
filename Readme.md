# Kramp Hub Example
[![Build Status](https://travis-ci.org/B-Stefan/Kramp-Hub-Example.svg?branch=master)](https://travis-ci.org/B-Stefan/Kramp-Hub-Example)

## Getting started 

### Running 

Go the the project root in your terminal and execute the following command: 

```
$ ./gradlew bootRun
```
 
### Testing 

```
$ ./gradlew test
```

## Requirements 

* Java 9
